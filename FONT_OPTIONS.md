# Font Options for Your Site

## Current Active Font: Option 1 - Editorial Elegance
A modern, bold combination that gives your site a contemporary editorial feel.

## How to Switch Fonts

To change the font style of your site, follow these simple steps:

### Step 1: Update HTML (index.html)
Uncomment the font option you want to use in the `<head>` section and comment out the current one.

### Step 2: Update CSS (css/bustling-world-v2.css)
In the CSS file, look for the Typography section (around line 28) and:
1. Comment out the current active font variables
2. Uncomment the font option you want to use

## Mobile Considerations

All font options are optimized for both desktop and mobile viewing. The Inter and Work Sans options provide the best readability on small screens, while Bebas Neue and Archivo Black create the most visual impact.

## Font Pairing Tips

- **High Contrast:** Pair serif subtitles with sans-serif headers for contrast
- **Consistency:** Use fonts from the same option for cohesive design
- **Readability:** Always prioritize body text readability over style
- **Character Selection:** The bold options (1, 3) work great with your character selection screen
- **Professional Pages:** Options 4 and current body font (Inter) work best for bio and writing pages
